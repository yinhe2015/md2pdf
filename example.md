# Markdown 转 PDF 示例文档

这是一个用于测试 **md2pdf** 工具的示例文档。

## 功能特性

- 支持 **粗体** 和 *斜体* 文本
- 支持中文排版
- 代码高亮
- 表格支持
- 引用块

## 代码示例

### Python 代码

```python
def hello_world():
    print("Hello, World!")
    return 42
```

### JavaScript 代码

```javascript
const greeting = (name) => {
    console.log(`Hello, ${name}!`);
};
```
